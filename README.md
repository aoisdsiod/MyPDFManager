## 功能

- 📄 **PDF 阅读** — 支持连续滚动和单页两种阅读模式，手势缩放、页面导航
- 🔖 **收藏管理** — 创建收藏夹，对 PDF 文件分类管理
- 🏷️ **标签系统** — 自定义标签和分类，多维筛选过滤
- 🔍 **全文搜索** — 文件名搜索 + 内容索引搜索
- 🔄 **格式转换** — PDF 转图片，支持页面选择
- 🖼️ **缩略图预览** — 自动生成 PDF 缩略图，快速浏览
- 📤 **分享与导出** — PDF 文件导出到其他应用或目录
- 📦 **ZIP 处理** — 压缩包内 PDF 文件的解压与管理
- ⚙️ **数据库管理** — 查看和管理本地缓存数据

## 详细使用说明

> 以下说明均基于实际操作流程编排，建议你打开 App 配合阅读。

### 首次启动

1. **选择 PDF 库文件夹** — 打开 App 后第一步是手动选择你存放 PDF 的文件夹
2. **自动扫描** — App 会自动读取该文件夹下的所有 PDF 文件
3. **缩略图** — 刚打开时卡片显示的是文件名，不要着急，进入设置页面，滑到最下方点击 **"生成缩略图"** 按钮。文件较多时请耐心等待，生成完成后会弹出提示（若失败可重试）
4. 缩略图生成完毕后，点击文件卡片即可进入对应 PDF 的**详情界面**

> 具体界面截图不便展示，请自行打开 App 体验。

---

### 🏷️ 标签系统

标签分为两个层级：**标签类别** 和 **标签**。

- 每个标签类别下可设置多个标签
- **管理入口**：设置页面 → 点击 **"管理标签"** 按钮
- **为文件添加/删除标签**：在 PDF 详情页面操作

---

### 📄 PDF 阅读

内置 PDF 阅读器，支持两种模式：

- **连续滚动模式** — 上下滑动阅读
- **单页模式** — 左右翻页阅读

> 两种模式下，**双击屏幕** 均可唤出工具栏，用于切换阅读模式和翻页

#### 阅读设置（设置页面 → 阅读设置）

可选择工具栏显示模式：

| 显示模式 | 说明 |
|---------|------|
| **正常显示** | 双击正常唤出工具栏，功能完整 |
| **仅显示页码** | 双击不唤出工具栏，仅保留左上方的页码显示（纯享模式） |
| **完全隐藏** | 连页码也不显示（真正的纯享模式） |

---

### 🔍 筛选功能

- 选择已有标签，支持 **且 / 或** 逻辑组合筛选
- 同一标签类别下的标签默认逻辑为 **或**
- **进阶用法**：选择好筛选条件后，点击 **"保存到收藏"** 按钮，可将该筛选条件固化到**收藏页面**
  - 之后在收藏页面直接点击即可查看当前数据库中符合条件的所有文件

---

### 🔖 收藏管理

- 在收藏页面可以**创建文件夹**，用于管理固化的筛选条件
- **长按卡片** 弹出交互面板
- **排序**：点击卡片右侧的按钮调整顺序

---

### 🔄 格式转换

> 该功能仅支持转换 **ZIP 压缩包**，压缩包内部要求以**规律数字命名的多个 WebP 文件**。

- **入口**：设置页面 → **PDF 转换** 卡片
- **扫描路径**：默认扫描库文件夹下的 `/database/zip` 文件夹，也可绑定额外扫描的文件夹
- **批量转换**：可选择同时转换多个 ZIP（默认转换全部图片）
- **单文件转换**：也可逐个转换，支持只选择部分页面

---

### 📤 分享与导出

**分享入口**：PDF 详情界面 / 设置页面的多选分享

- 将选中的 PDF 文件复制到指定文件夹
- 目标文件夹限定在：`库文件夹/database/share/` 及其一级子目录

**导出分享流程**：

1. 点击 **"导出分享"** 按钮
2. 选择目标文件夹
3. App 扫描该文件夹中的 PDF 文件，与数据库进行对比
4. 输出数据库中有记录的文件的：**名称、标签、备注** 等信息
5. 最终生成一个 `txt` 文件，也可选择复制到剪贴板

---

### ⚙️ 数据库管理

> 用于查看和管理本地缓存数据（数据库信息）。每个数据库对应一个库文件夹。

#### 导出数据库

直接选择导出后的存放路径即可。

#### 导入数据库

1. 第一次文件选择：选择要导入的数据库文件
2. 第二次文件夹选择：选择要绑定的文件夹

**数据库会保存**：PDF 信息、标签类别与标签、每个 PDF 的标签和备注、收藏页面的所有项、设置信息

**数据库不会保存**：缩略图，具体PDF的文件数据

#### ⚠️ 重要提醒

> 若你从别人那里导入数据库，但本地没有相应的 PDF 文件，**切勿轻易点击 "扫描库文件夹" 按钮**—— 那样会直接删除数据！

- 导入他人数据库后，你可以看到对方对每个 PDF 的设置和评价
- 但无法查看 PDF 的缩略图和文件本身
- 建议：根据文件名自行查找对应文件，下载并转换后即可在 App 内正常浏览
